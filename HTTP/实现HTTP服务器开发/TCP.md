## TCP

![51](../../photo/54.jpg)

### 可靠传输

-   保证数据的准确传输

### 流量控制

-   感知对方压力控制流量
-   如服务器与普通计算机之间进行数据传输，由于网卡性能差异，通过TCP协议可以感知到，服务器会控制流量

### 拥塞控制

-   感知网络压力并控制发送速度，互联网压力

### 套接字

-   {IP : Port}  –>  套接字       （计算机IP,   进程Port）
-   套接字（Socket）是抽象概念，表示TCP 链接的一端
-   通过套接字可以进行数据发送或接收

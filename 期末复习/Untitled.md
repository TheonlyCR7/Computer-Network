```
当数据帧为 128 字节时
发送时延为 = (128 X 8) / 16000 = 0.064 s
总时延 = (0.064 + 0.27) X 2 = 0.668 s
因为使用的是GBN 所以在第一个帧发送确认前，会一直有帧发出
在 0.668 s 内    0.668 / 0.064 = 10.4 帧
2^n - 1 = 10.4 帧  加上一帧来区别新旧
n 至少为 4
当数据帧为  512 字节时   同理    但得到的帧数比前者小
```
